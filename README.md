# TXT Wallpaper

**React Native, Expo, Typescript**

App mobile desenvolvido com React Native e Expo, que disponibiliza wallpapers.

**Tecnologias utilizadas:**

- React Native
- Expo
- Typescript
- react-native-google-mobile-ads
- react-native-reanimated
- styled-components
